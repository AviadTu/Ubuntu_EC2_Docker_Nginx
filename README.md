\# AWS Docker Nginx Lab



\## Overview

Created an Ubuntu EC2 instance on AWS, installed Docker, and deployed an Nginx web server inside a Docker container.



\## Steps Completed

\- Launched EC2 Ubuntu instance (Free Tier)

\- Configured Security Group (SSH 22, HTTP 80)

\- Connected via SSH

\- Installed Docker Engine

\- Ran Nginx container using Docker

\- Mapped port 80 to the public internet

\- Verified access via browser using public IP



\## Result

Nginx default page is accessible via EC2 public IP.



\## Cleanup

Stopped and removed Docker container and terminated EC2 instance.
